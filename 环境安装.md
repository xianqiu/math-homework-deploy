# 前端

## node

```
# 安装 node 16
dnf module install nodejs:16

# 升级 npm
npm install -g npm

# 清除npm缓存
npm cache clean -f

# 安装n模块
npm install -g n

# 升级node.js到最新稳定版
n stable
```

